PROGRAMMING PROJECT 3: TEXTLINE
Creator: Bryan Liang
Date: July 3, 2017

INTRODUCTION:
TextLine is a program to represent a single line of text (characters) with an array of characters.
Methods are provided to manipulate and edit the TextLine objects.

REQUIREMENTS:
This program requires an up-to-date installation of Java. A UNIX terminal or an internal 
development environment is also necessary.

CONFIGURATION:
The main contents of the program are stored in the following package: edu.uga.cs1302.txtbuff
The application to run the program is found in the following package: edu.uga.cs1302.test
The program's documentation is found within the /doc/ folder in this directory.

To compile the Java files, execute the following UNIX command in the parent directory of the package.
	$ javac -d classes edu/uga/cs1302/txtbuff/*.java
	$ javac -d classes edu/uga/cs1302/test/*.java
Next start up the application with the following command:
	$ java edu.uga.cs1302.test.TextLineTester
